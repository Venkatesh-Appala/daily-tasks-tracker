# Bujjamma Daily Tasks Tracker (SPA)

Simple single-page app scaffold (React + Vite + PWA).

## Quick start

Install dependencies:

```bash
npm install
```

Run dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Notes
- The app uses `localStorage` by default. Add a backend for syncing across devices.
- Add icons at `public/icons/icon-192.png` and `public/icons/icon-512.png` for PWA install.
